# smartCampusPlatform

#### 介绍
基于Springboot实现的校园综合服务平台
本系统实现了以校园生活为中心的学科竞赛、校园兼职、失物招领等相关信息服务，为师生的校园生活提供一个便捷的服务平台。

#### 软件架构
前端：VUE+Element UI
后端：SpringBoot+SSM
